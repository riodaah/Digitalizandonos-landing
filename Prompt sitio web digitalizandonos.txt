🧠 Prompt Maestro – Digitalizandonos 2.0 (para Cursor)

Crea una landing page moderna y animada para la empresa Digitalizándonos, usando React, TailwindCSS y Framer Motion.
El sitio debe ser de una sola página con efectos visuales atractivos tipo https://www.jeton.com/es
.

⚙️ Stack y Arquitectura

React + Vite

TailwindCSS

Framer Motion (para animaciones de scroll, fade-in, parallax y movimiento dinámico)

React Router (solo si se requiere manejar popups de políticas como rutas modales)

Archivo config.json en la raíz para parametrizar:

Colores primarios, secundarios, gradientes

Tipografía global (Orbitron por defecto)

Textos clave (nombre agente, teléfono, dirección, CTA, etc.)

Enlaces de WhatsApp, correo y redes sociales

🎨 Estilo general

Modo oscuro permanente

Fondo degradado negro → morado profundo, con acentos amarillo oscuro y toques neón.

Usa Orbitron para títulos y Poppins para párrafos.

Textos grandes, legibles, minimalistas y con “startup energy”.

Efectos suaves de scroll y hover tipo Jeton (motion.div + parallax + opacidad dinámica).

Gradientes animados sutiles en botones y encabezados.
